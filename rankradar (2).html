Project Abstract

RankRadar — AI-Powered
India College Finder

Overview: RankRadar is an intelligent, single-page web application designed to help Indian students navigate the complex and often overwhelming process of college selection after competitive entrance examinations.

Problem Statement: Each year, millions of Indian students appear for entrance exams like JEE, NEET, CAT, CLAT, GATE, and state-level tests. Choosing the right college from thousands of options — while balancing rank eligibility, fees, location, course preferences, and career priorities — is a highly stressful multi-variable decision that most students and families are ill-equipped to make alone.

Solution: RankRadar combines a structured local college database (curated with NIRF rankings, fee data, placement statistics, and eligibility cut-offs) with a live AI agent powered by Claude to deliver personalised, ranked college recommendations in seconds. The system pre-filters colleges by exam compatibility, rank bracket, budget ceiling, and geographic preference, then scores each institution using a weighted multi-factor algorithm before presenting the results with AI-generated contextual explanations tailored to each student's profile.

Key Features

Support for 25+ entrance exams across engineering, medical, law, management and postgraduate disciplines. Dynamic rank/percentile parsing. Budget slider with real-time annual fee filtering. Category-based reservation adjustments (SC/ST/OBC/EWS). Priority weighting across placements, research, campus life, fees, and NIRF rank. AI-generated personalised summaries and actionable pro tips per student profile.

Architecture

Built as a fully self-contained HTML/CSS/JavaScript single-page app with zero backend dependencies. Authentication is handled client-side (demo mode) with session state stored in memory. The AI agent communicates directly with the Anthropic Claude API from the browser for real-time college analysis.

Tech Stack

HTML5 CSS3 (Custom Properties) Vanilla JavaScript Claude API (Sonnet) DM Sans & Syne Fonts NIRF 2024 Data

Target Users

Class 12 students and recent graduates appearing for national and state-level entrance examinations in India. The tool is particularly valuable for first-generation college students who lack access to professional counselling services.
